What's Included in zola-astroplate

zola-astroplate is a comprehensive starter theme that includes everything you need to get started with Zola.
  • 10+ pre-built pages
  • Tailwind CSS v4 integration
  • Alpine.js for interactivity
  • Fully responsive on all devices
  • SEO-optimized
  • Open-source and free

Discover the Key Features Of Zola

Zola is a fast, all-in-one static site generator. It offers a range of exciting features for developers and creators.
  • Single executable, no dependencies
  • Extremely fast builds
  • Tera templates (Jinja2-like)
  • Built-in search and taxonomies
